Residual size S(f) under each attack strategy

The shaded band shows the mean ± standard deviation of the same curves on the weight-rewiring null model. A curve below the band indicates a strategy that fragments the network faster than chance. The weighted-efficiency chart tracks how well the surviving core stays knit (Latora-Marchiori efficiency, sampled on a coarse grid) — it can collapse while the residual-size curves still look healthy.

Ban-wave scenarios

Residual size after removing each whole community in one step, next to the damage that removing the same number of channels uniformly at random would cause (rnd). A red value sits below its random baseline: banning that block damages the network more than its size alone explains — a load-bearing sub-ecosystem. Values at or above the baseline mark blocks the network could absorb losing, however large.

Modular robustness

Fraction of intra-community vs inter-community edges surviving the attack, normalised to the q = 0 baseline. Strategies that strip inter-community ties first (red below blue) tend to decouple sub-ecosystems before eroding individual community cohesion.
